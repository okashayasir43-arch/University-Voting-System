# University Online Voting System
A secure online voting platform for university elections with advanced biometric authentication.

## Features

### User Features
- Dual Authentication System
- Face Recognition Login (biometric)
- Traditional Password Login
- One-time voting per election
- Real-time election results
- User dashboard

### Admin Features
- Create and manage elections
- Edit election dates
- Add candidates
- View comprehensive results
- Create admin accounts

## Technology Stack

- **Backend**: ASP.NET Web Forms (.NET Framework 4.7.2)
- **Database**: SQL Server
- **Frontend**: HTML, CSS, JavaScript
- **Face Recognition**: face-api.js library

## Security Features

- Role-based access control
- Database-level one-vote constraint
- SQL injection prevention
- Session management
- Biometric authentication
